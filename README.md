Loot Notifier

A RuneLite plugin that displays a collection log style popup when you receive a valuable drop.

Features
- 4 configurable loot value tiers (Low, Medium, High, Insane)
- Custom GP threshold per tier
- Custom header text per tier
- Custom border colour per tier with full colour picker
- Authentic OSRS collection log style popup
- Smooth expand and collapse animation

Configuration
Open the plugin settings to configure each tier:
- **Enable** — toggle the tier on or off
- **GP threshold** — minimum item value to trigger the popup
- **Header text** — text displayed in the popup header
- **Border colour** — colour of the popup border
